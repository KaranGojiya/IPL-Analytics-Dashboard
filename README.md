# 🏏 IPL Analytics Dashboard (Python + Tkinter)

This project is a data analysis dashboard built using Python.

It analyzes IPL data using:
- Pandas for data processing
- Matplotlib for visualization
- Tkinter for GUI dashboard

---

## 📊 Features

- 🟠 Orange Cap (Top Run Scorers)
- 🟣 Purple Cap (Top Wicket Takers)
- 🔵 Most Sixes
- 🟢 Most Fours
- 📈 Team Win Percentage
- 🎯 Most Dot Balls
- ⚡ Economy Rate Analysis

---

## 🧰 Tech Stack

- Python
- Pandas
- Matplotlib
- Tkinter

---

## 🖥️ How to Run

```bash
pip install pandas matplotlib
python app.py
```
